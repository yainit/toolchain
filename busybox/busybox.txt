Please download the current BusyBox master file
from  http://www.busybox.net


For to build an own set of busybox binaries, extract the archive
of busybox source and copy the scripts in this folder.
With the current binary from init/initramfs/bin you can extract 
the .config file and modify according your wants.  

Use update.sh to create DOT.config files and make.sh to compile
them in a single run. 
Use the patch before, if you need real use of -march from your 
.config file and if you want recognize  VFAT32 partitions on 
block devices greater or equal than 1TB.
